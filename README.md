  ## Cómo inicializar el proyecto
  Ejecutar el comando:
  ```cmd
pnpm install
```
y luego
  ```cmd
pnpm run migrate
```

Finalmente se inicia el puerto con
  ```cmd
pnpm run start
```
o en producción

  ```cmd
pnpm run open
```

## Requisitos

 - Node.js 18+
 - pnpm habilitado
 - Se debe añadir las credenciales de una base de datos en Turso en un archivo **".env"**
